NeuroAI follows a futuristic dark-themed aesthetic inspired by brands like OpenAI and Apple. Key design elements include:

Glassmorphism: Frosted glass effects on cards and navigation.

Neon Glows: High-contrast purple and blue gradients to emphasize AI "energy."

Motion Design: Smooth, scroll-triggered animations and fluid transitions.

✨ Key Features
Dynamic Hero Section: Featuring a custom-coded Typing Effect and an animated HTML5 Canvas particle background.

Interactive Chatbot Demo: A simulated AI chat interface with realistic message delays and typing indicators.

Dark/Light Mode: A fully functional theme switcher that remembers user preference using localStorage.
